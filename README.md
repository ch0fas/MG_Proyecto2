# MG_Proyecto2
This project maps all* of the world's airports, using centrality metrics and path-finding algorithms to answer various queries. 

We also propose two different "tour" vacation packages, using graph techniques to find the most interesting destinations and most efficient routes between said destinations.

This project was worked on by **Sofia Maldonado García** and **Viviana Toledo de la Fuente** during our Graph Mining class at ITESO in the Summer of 2025.
